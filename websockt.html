3.2. 获取K线

订阅topic: future_kline
消息订阅格式如下所示：

{
    "header": {
        "type": 1003
    },
    "body": {
        "topics": [
            {
                "topic": "future_kline",
                "params": {
                    "symbols": [
                        {
                            "symbol": 100,
                            "ranges": [
                                "60000"
                            ]
                        }
                    ]
                }
            }
        ]
    }
}

1、支持按照1分钟5分钟这种单独订阅，需要订阅所有，就传数组，一定要传
2、lines 的数据结构为：

[
    [${时间戳}, ${开市价格}, ${最高价格}, ${最低价格}, ${闭市价格}, ${成交量}]
]

3、range取值：

  • 1Min = 60000ms

  • 3Min = 180000ms

  • 5Min = 300000ms

  • 15Min = 900000ms

  • 30Min = 1800000ms

  • 1Hour = 3600000ms

  • 2Hour = 7200000ms

  • 4Hour = 14400000ms

  • 6Hour = 21600000ms

  • 12Hour = 43200000ms

  • 1Day = 86400000ms

  • 1Week = 604800000ms
